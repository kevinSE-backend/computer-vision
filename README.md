# computer-vision
CNN that classifies images of cats and dogs

This project is all about coming up with a CNN that classifies different cats and dogs into 37 classes by breed type.

## files

## folders.py ##
this folder contains python script that creates subfolders to train and val for cats and dogs so that the classify.py file can access the data in a more organised way.

## classify.py ##
Entails the CNN python code that uses torch module

## pretrained-model.py ##
Entails loading a pretrained network

## requuirements.txt ##
all modules and packages used in this project
